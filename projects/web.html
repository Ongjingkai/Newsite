Web Development


HTML & CSS

HTML
The standard markup language for Web pages is HTML (Hypertext Mark-Up Language). You can make your own website using HTML. It also allows users to create links (points) to other websites, materials, photos, video, and audio.
To learn more about HTML visit Jake Wright's video.
CSS
The language that we use to style an HTML document is CSS. CSS specifies how HTML elements should appear. This is used to make the website more organized and user-friendly.
To learn more about CSS visit Jake Wright's video.
